# 2026-07-13

## RPC 客户端 API 重命名

- `NotifyAsync` → `InvokeActionAsync`：单向通知，对应 C# `Action` 无返回值语义
- `InvokeAsync` → `InvokeFuncAsync`：请求-响应，对应 C# `Func<T>` 有返回值语义
- 所有文件中 "强制响应" → "必须响应"
- 更新注释使其直观易懂（如 "Func 语义，有返回值，必须等待结果" / "Action 语义，无返回值，发射后即忘"）
- 影响文件：RpcClientBase.cs, README.md, RPC 服务框架.md, 远程过程调用(XML-RPC)消息协议.md, MEMORY.md
