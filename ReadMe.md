## SpaceCG 常用的标准代码库

- ### SpaceCG.Standard 
	- 基于 .Net Standard 2.0 的标准代码库
	- 基础代码库，包含一些基础的工具类，以及一些基础的数据结构
	- 支持 WPF、U3D 项目使用	
	
- ### SpaceCG.Standard.WPF
	- 基于 .Net WorkFramework 4.8 的标准代码库
	- 包含 窗体、UI等扩展
