## SpaceCG 常用的标准代码库

- ### Builder 里面都是构建工程
- ### SharedProjects 里面都是代码

---

- ### SpaceCG.Standard 
	- 基于 .Net Standard 2.0 的标准代码库
	- 基础代码库，包含一些基础的工具类，以及一些基础的数据结构
	- 支持 WPF、U3D 项目使用	
	- 不使用任何第三方库，仅使用 .Net 自带的库，为跨平台、跨工具使用提供支持
	
- ### SpaceCG.Standard.WPF
	- 基于 .Net Framework 4.8 的标准代码库
	- 包含 窗体、UI等扩展
	
- ### SpaceCG.Standard.U3D 
	- 基于 .Net Framework 4.8 的标准代码库

- ### SpaceCG.Standard.Device 
	- 基于 .Net Framework 4.8 的标准代码库

- ### SpaceCG.Standard.LedDevice 
	- 基于 .Net Framework 4.8 的标准代码库
